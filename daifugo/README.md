# 大富豪 — Daifugo

4人対戦のトランプ「大富豪」。あなた + CPU 3人(松・竹・梅)で対戦。
`index.html` をブラウザで開くだけで遊べる(ビルド不要・依存ゼロの1ファイル構成)。

## ルール

- 3が最弱、2が最強(3 < 4 < … < K < A < 2)
- 同じ数字は 2〜4 枚同時に出せる
- 階段(同じスートの連続数字)は 3 枚以上で出せる
- 複数枚で始まった場は同じ枚数・同じ種類でしか返せない

### 特殊ルール

| ルール | 内容 |
|---|---|
| ジョーカー | すべてに勝る。複数枚プレイのワイルドカードとしても使える |
| スペ3返し | ジョーカー単体に勝てるのは ♠3 だけ。出されたら場が流れる |
| 8切り | 8 を含むプレイで場が流れ、出した人がリード |
| Jバック | J(11)を含むプレイで、場が流れるまで序列が一時逆転 |
| 革命 | 4枚以上の同時出しで序列が逆転。革命中の革命で元に戻る |

革命と Jバックが重なった場合は打ち消し合って通常の序列に戻る。

## 操作

カードをクリック(タップ)で選択 → 「出す」。出せる手がないときは「パス」。
最初に手札を出し切った人から 大富豪 / 富豪 / 貧民 / 大貧民。
