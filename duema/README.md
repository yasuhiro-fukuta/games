# トランプデュエル — duema

トランプ2デッキで遊ぶデュエマ風カードバトル。あなた(♦13+♥13+JOKER) vs CPU(♣13+♠13+JOKER)。
`index.html` をブラウザで開くだけで遊べる(ビルド不要・依存ゼロの1ファイル構成)。

## ルール

- デッキ27枚、シールド6枚、初手5枚(先攻1ターン目はドローなし)
- **手札は土地としてもクリーチャーとしてもプレイできる**
  - 土地は1ターンに1枚。タップでそのスートのマナ1を出す
  - クリーチャーのコスト = 自スート1マナ + 数字ぶんの無指定マナ(例: ♠3 → 合計4マナ)
  - パワー/タフネス = 数字。出したターンは攻撃できない(召喚酔い)
- **攻撃**: 相手プレイヤー(シールドブレイク)or タップ中の敵クリーチャー。
  防御側はアンタップのクリーチャーでブロックできる(ブロックするとタップ)
- **スート相性**(戦闘時、守り側に +0/+1):
  - ♥ vs ♠ → ♠ が +0/+1
  - ♦ vs ♠ → ♦ が +0/+1
  - ♦ vs ♣ → ♣ が +0/+1
  - ♥ vs ♣ → ♥ が +0/+1
- **シールドトリガー**: ブレイクされたカードは手札へ。そのままコストなしで召喚してもよい
- **JOKER**: 4色どのマナも出せる土地、または「0マナ: すべてのクリーチャーを破壊」の呪文
- **勝利条件**: シールドが0の相手に攻撃を通す(ダイレクトアタック)。山札が尽きたら負け
- **後攻補正**: 後攻は1回だけフリーマリガン(手札5枚を山札に戻してシャッフルし、5枚引き直し)できる

## 操作

手札をクリック → 土地に置く / 召喚を選択。自分のクリーチャーをクリック → 攻撃対象(相手エリア or タップ中の敵)をクリック。「ターン終了」で相手番へ。
